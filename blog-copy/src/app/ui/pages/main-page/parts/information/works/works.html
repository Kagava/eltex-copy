Мои работы

Все три проекта созданы на чистом стеке технологий без фреймворков: HTML5, CSS3 и ванильный JavaScript. Базовая вёрстка во всех случаях семантическая и адаптивная, с поддержкой отображения на экранах от 320 до 1280+ пикселей. Проект Shelter фокусируется на точной вёрстке по макету, методологии БЭМ и доступности. Christmas Shop добавляет интерактивность на чистом JS: динамическую генерацию контента, работу корзины и сохранение данных в localStorage. Pair 'em Up — наиболее сложный проект с полноценной игровой логикой, управлением состоянием, алгоритмами проверки пар чисел, обработкой событий и аудиоэффектами. Во всех работах применялись современные инструменты разработки (Vite, ESLint, Prettier), деплой через GitHub Pages, оптимизация изображений в WebP, а также требования к кроссбраузерности и доступности. Проекты демонстрируют постепенное усложнение: от статичной вёрстки до интерактивного приложения с игровой механикой, сохраняя фокус на фундаментальных веб-стандартах.

Winter 2026

Pair'em Up

Pair 'em Up — это интерактивная браузерная игра-головоломка, где игрок должен находить и удалять пары чисел с игрового поля, набирая 100+ очков. Проект направлен на отработку навыков ванильного JavaScript, работы с DOM, управления состоянием и создания адаптивного интерфейса.

Summer 2024

Cozy House

Разработать адаптивный одностраничный сайт (лендинг) для приюта домашних животных «Cozy House». Проект направлен на отработку навыков вёрстки на HTML5 и CSS3, работы с макетом, создания анимаций и адаптивного дизайна.

Autumn-Winter 2024

Chirstmas Shop

Christmas holidays are just around the corner! It's time to shop for gifts... but where? Christmas Shop is a project where you will need to create a website consisting of two pages, make it responsive, and add interactivity. It's time to create a cozy place for buying interesting gifts!
